# 🏛️ Gujarat Sarkari Naukri Portal

A free, auto-updating Gujarat Government Jobs website.

## 🌐 Live Website
👉 [Click here to open](https://YOUR-USERNAME.github.io/gujarat-jobs/)

## 📋 Features
- Auto-refreshes every 5 minutes
- 24+ government departments (GPSC, GSSSB, Police, Health, Education...)
- Search, filter by department & job class
- Deadline countdown on every job
- Direct links to official apply portals

## 🏢 Departments Covered
GPSC · GSSSB · OJAS Gujarat · Gujarat Police · Health Dept · Education Dept · Revenue Dept · PWD · Forest Dept · Agriculture · Finance · GUVNL · Municipal Corp · Panchayat

## 📁 How to Run Locally
1. Download `index.html`
2. Double-click the file
3. It opens directly in your browser — no setup needed

## 🚀 How to Deploy on GitHub Pages
1. Upload `index.html` to your GitHub repo
2. Go to Settings → Pages
3. Set source to `main` branch → Save
4. Site is live in ~2 minutes
